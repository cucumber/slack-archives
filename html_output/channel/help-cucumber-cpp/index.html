paoloambrosio
2018-04-26 06:46:25

tmp\test_features\step_definitions\cpp_steps.cpp does not exist in the repository. End-to-end tests in Cucumber-CPP need to generate the step definitions to be compiled. That file is generated at runtime by those E2E tests and is referenced by CMake so that they can be compiled in each scenario. You probably don't need to run those E2E tests unless you want to submit contributions to the project. You can easily disable them with -DCUKE_DISABLE_E2E_TESTS=on
